# EdgeTranslate-v3

EdgeTranslate-v3 — это браузерное расширение-переводчик для Manifest V3, основанное на
оригинальном проекте [Edge Translate](https://github.com/EdgeTranslate/EdgeTranslate).
Этот форк сохраняет привычный сценарий перевода выделенного текста и одновременно
адаптирует расширение к современным правилам Chrome, Firefox и Safari.

Линейка 4.x сосредоточена на обновленном интерфейсе в духе Material Design, глубоко
обновленном просмотрщике pdf.js, темной теме, более удобной панели перевода и
экспериментальном локальном AI-переводе Chrome через Gemini Nano там, где браузер его поддерживает.

- Текущий репозиторий: [Meapri/EdgeTranslate-v3](https://github.com/Meapri/EdgeTranslate-v3)
- Оригинальный проект: [EdgeTranslate/EdgeTranslate](https://github.com/EdgeTranslate/EdgeTranslate)
- Последний релиз: [v4.0.1](https://github.com/Meapri/EdgeTranslate-v3/releases/tag/v4.0.1)

## Основные возможности

- Перевод выделенного текста в боковой панели с копированием, редактированием, закреплением,
  изменением размера и озвучиванием.
- Помощь со словами: перевод, подробные значения, определения и примеры, если выбранный
  провайдер это поддерживает.
- Поддержка встроенного AI-перевода Chrome для обычного текста и сценариев перевода слов.
- Google и другие практичные движки остаются доступными для перевода страниц целиком.
- Встроенный просмотрщик pdf.js с переводом выделенного текста внутри PDF.
- PDF-панель, меню, диалоги, боковая панель страниц и темная тема в стиле Material.
- Темная тема для PDF-просмотрщика, панели перевода, всплывающего окна и страницы настроек.
- Очищенная страница настроек с более понятными названиями провайдеров и меньшим визуальным шумом.
- Горячие клавиши, черные списки и настройка поведения кнопки перевода.

## Заметки об AI-переводе

Локальный путь Gemini Nano в Chrome отображается как AI-провайдер перевода, когда браузер
предоставляет `LanguageModel` API. Он полезен для коротких выделенных фрагментов и помощи со
словами, но в этом форке не используется как движок перевода страниц целиком.

Важные особенности:

- Перевод страниц через Gemini Nano удален из пользовательских настроек перевода страниц,
  потому что текущая локальная производительность пока не подходит для полноценного перевода страниц.
- Обычный AI-перевод ограничен по параллельности, чтобы сохранить разумный баланс между
  отзывчивостью и температурой CPU.
- Вывод промпта разбирается защитно, чтобы некорректный JSON не попадал прямо в панель перевода.
- Частые непереведенные фрагменты из AI-вывода очищаются без медленного второго прохода модели.
- Видимый текст произношения удален из карточек результата; озвучивание TTS сохранено.

Доступность `LanguageModel` API и Gemini Nano зависит от версии Chrome, устройства, флагов функций
и состояния загрузки локальной модели. Если AI-перевод недоступен, расширение возвращается к
настроенному провайдеру и не считает AI обязательной зависимостью.

## PDF-просмотрщик

EdgeTranslate-v3 открывает поддерживаемые PDF-ссылки во встроенном просмотрщике pdf.js, чтобы
выделение текста и перевод работали внутри документов. Просмотрщик 4.x использует pdf.js `5.7.284`
и содержит множество исправлений макета и взаимодействия.

Поведение PDF:

- Web-ссылки на PDF могут перенаправляться в просмотрщик расширения для перевода.
- Локальные PDF можно открывать, если в Chrome для расширения включен доступ к файловым URL.
- Перетаскиваемые PDF-файлы и blob-ссылки просмотрщика предварительно загружаются или исправляются,
  чтобы pdf.js мог надежно их открыть.
- В просмотрщике есть действие обхода, позволяющее открыть текущий документ вне EdgeTranslate.
- Запасной механизм обнаружения PDF не проверяет нерелевантные не-PDF страницы, что предотвращает
  лишние CORS-ошибки на страницах вроде консоли разработчика Chrome Web Store.

## Поддержка браузеров

### Chrome

- Перевод выделенного текста
- PDF-просмотрщик и перевод выделенного текста в PDF
- Перевод страниц через Google
- Встроенный AI-перевод Chrome, если браузер и устройство поддерживаются
- Поддержка offscreen document для AI-перевода в контексте расширения

### Firefox

- Перевод выделенного текста
- PDF-просмотрщик и перевод выделенного текста в PDF с ограничениями конкретного браузера
- Нет встроенного AI-перевода Chrome
- Нет поведения перевода страниц, доступного только в Chrome

### Safari

- Перевод выделенного текста и поддержка PDF-просмотрщика через сборку Safari extension
- Нет встроенного AI-перевода Chrome
- Для релиза Safari нужны Xcode-проект и учетные данные Apple

## Среда разработки

Работайте из корня репозитория.

```bash
npm ci
```

Запустить набор unit-тестов:

```bash
npm test
```

Запустить тесты рабочего пространства EdgeTranslate напрямую:

```bash
npm test -w edge_translate -- --runInBand
```

## Команды сборки

Собрать цель Chrome по умолчанию:

```bash
npm run build:chrome
```

Собрать отдельные цели:

```bash
npm run build:chrome
npm run build:firefox
npm run build:safari
```

Создать браузерные пакеты:

```bash
npm run pack:chrome -w edge_translate
npm run pack:firefox -w edge_translate
```

Проверить пакет Firefox:

```bash
npm run lint:firefox
```

Выходные файлы сборки:

- Распакованная сборка Chrome: `packages/EdgeTranslate/build/chrome/`
- Распакованная сборка Firefox: `packages/EdgeTranslate/build/firefox/`
- Пакет Chrome: `packages/EdgeTranslate/build/edge_translate_chrome.zip`
- Пакет Firefox: `packages/EdgeTranslate/build/edge_translate_firefox.xpi`
- Вывод сборки Safari: `packages/EdgeTranslate/build/safari/`

## Загрузка локальных сборок

### Chrome

1. Откройте `chrome://extensions`.
2. Включите режим разработчика.
3. Нажмите "Load unpacked".
4. Выберите `packages/EdgeTranslate/build/chrome/`.
5. Включите "Allow access to file URLs", если нужен перевод локальных PDF или файлов.

### Firefox

1. Откройте `about:debugging`.
2. Выберите "This Firefox".
3. Нажмите "Load Temporary Add-on".
4. Выберите любой файл внутри `packages/EdgeTranslate/build/firefox/` или используйте созданный
   `edge_translate_firefox.xpi`.

### Safari

Сборки Safari находятся в `packages/EdgeTranslate/safari-xcode/`.

Полезные команды:

```bash
npm run build:safari
npm run safari:sync -w edge_translate
npm run safari:release -w edge_translate
```

`safari:release` собирает проект, синхронизирует ресурсы в Xcode, архивирует, экспортирует и
загружает сборку. Для этого нужны действительные учетные данные App Store.

## Разрешения

Расширение использует доступ к хостам, чтобы перевод выделения, обнаружение PDF и content scripts
работали на страницах пользователя. Chrome-сборки также запрашивают `offscreen`, чтобы Gemini Nano
мог выполняться из контекста документа расширения, когда внедрение в страницу недоступно, например
во встроенном PDF-просмотрщике расширения.

Основные категории разрешений:

- `activeTab`, `tabs` и `scripting` для перевода выделения и взаимодействия со страницей.
- `contextMenus` и `storage` для команд и пользовательских настроек.
- `webNavigation` и `webRequest` для обнаружения PDF и маршрутизации просмотрщика.
- `offscreen` в Chrome для AI-перевода в контексте расширения.

## Конфиденциальность

- Этот форк не добавляет аналитику или сбор телеметрии.
- Текст для перевода отправляется только выбранному или настроенному пользователем провайдеру.
- Локальный AI-перевод Chrome, когда он доступен, выполняется через встроенный локальный путь модели Chrome.
- Доступ к файловым URL необязателен и управляется настройками расширения в браузере.
